# Special Sauce

A single-file deck of game-feel ingredients and design constraints — built to make your game weirder, juicier, and more memorable.

## What it's for

**Vibe-coded games and game jams.** You've got a prototype that plays fine but feels flat. Special Sauce gives you a concrete thing to add — a specific touch, a twist, a rule that forces the mechanic to become yours. It's the fastest way to go from "it works" to "wait, this is actually fun."

Draw a random **ingredient** (hitstop, coyote time, flavor text, a useless honk button) and a **constraint** (make it cost something, subvert it, put it in the world) and you've got an instant design prompt. The combo mode pairs them for you.

## Usage

No install. No build. Just open `index.html` in a browser.

```
open index.html
# or
python3 -m http.server 8080
```

## Modes

| Mode | What it does |
|------|-------------|
| **◆ Ingredients** | Draw a single game-feel touch to add |
| **▦ Constraints** | Draw a design rule that makes the feature unforgettable |
| **✦ Combo** | Pairs an ingredient + constraint as one design prompt — hit "↻ New twist" to reroll just the constraint |

Filter by category, search by keyword, or toggle **Goldmines** to surface high-impact, low-effort picks only.

## Game jam tip

Hit Space to draw. Keep hitting it. When one snags your attention, flip to Combo mode and see what constraint bends it into something no other jam entry will have.

There's also a secret card. The deck is about easter eggs — of course there is.

## One file

It's all in `index.html`. Send it to a teammate, fork it, steal the data arrays, add your own cards. Built to be raided, not finished.
